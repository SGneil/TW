題目 程式碼

西元 2013 年的 1 月 1 日為星期二，請撰寫程式從資料檔 d.txt 讀入兩個數字(中間以空格區分)，分別代表哪一個月的哪一天，接著算出此日期在 2013 年中為星期幾，最後將星期幾以數字輸出，星期日為 0，星期一為 1，星期二為 2，以此類推。

範例輸入一：
3 14
範例輸出一：
3 14
Ans=4
範例輸入二：
12 7
範例輸出二：
12 7
Ans=6

(0)
範例輸入：
12 7
範例輸出：
12 7
Ans=6

(1)
範例輸入：
2 15
範例輸出：
2 15
Ans=5

(2)
範例輸入：
11 11
範例輸出：
11 11
Ans=1

(3)
範例輸入：
10 27
範例輸出：
10 27
Ans=0

 
#include <stdio.h>
int main(void) {
    // 讀入輸入資料
    int now_month, now_day, total_day = 1, ans;
    scanf("%d %d", &now_month, &now_day);

    // 計算總天數
    total_day += now_day;
    for (int i = 1; i < now_month; i++) {
        if (i == 2) {
            total_day += 28;
        }
        else if (i <= 7) {
            total_day += 30 + i % 2;
        }
        else {
            total_day += 31 - i % 2;
        }
    }

     // 計算星期幾
    ans = total_day % 7;

    // 印出答案
    printf("%d %d\n", now_month, now_day);
    printf("Ans=%d\n", ans);
    return 0;
}
