題目 程式碼

請撰寫一個程式從檔案 e.txt 中讀入二個正整數 n 與 k(n<=100000000)，接著輸出 0~n的數值中，每個數字總合為 k 的個數。

範例輸入：
100 5
表示要從 0~100 中找出數字總合為 5 的數值個數，這些數值有：
5
14（因為 1 + 4 = 5）
23（因為 2 + 3 = 5）
32（因為 3 + 2 = 5）
41（因為 4 + 1 = 5）
50（因為 5 + 0 = 5）
範例輸出：
100 5
Ans=6

(0)
範例輸入：
100 5
範例輸出：
100 5
Ans=6

(1)
範例輸入：
100000000 2
範例輸出：
100000000 2
Ans=36

(2)
範例輸入：
78951384 11
範例輸出：
78951384 11
Ans=31648

 
#include <stdio.h>

// 確認傳入的個數相加是否等於k
int count_check(int num, int k) {
    int sum = 0;
    while (num != 0) {
        sum += num % 10;
        num /= 10;
    }
    if (sum == k) {
        return 1;
    }
    return 0;
}

int main(void) {
    // 讀入輸入資料
    int n, k, ans = 0;
    scanf("%d %d", &n, &k);

    // 計算範圍內有多少加起來等於k的數量
    for (int i = 0; i <= n; i++) {
        if (count_check(i, k)) {
            ans++;
        }
    }

    // 印出答案
    printf("%d %d\n", n, k);
    printf("Ans=%d\n", ans);
    return 0;
}
