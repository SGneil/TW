請從資料檔 d.txt 讀入好幾行字串資料
字串資料是由字元(‘a’~’z’,’A’~’Z’)、數字(‘0’~’9’)以及空白字元所構成的，接著將字串中這些由字元或者空白隔開的數字部分進行加總(54+256+72+125+56+22+66=751)，並且將原字串以及結果輸出。

#include <stdio.h>
#include <stdlib.h>

int main(void) {
    char str[201];
    int len = -1, sum = 0, temp_sum = 0;
    while (scanf("%c", &str[++len]) != EOF);  // 讀入輸入資料
    str[len] = '\0';        // 設定結束位置，有這個用%s輸出資料不會有亂碼

    for (int i = 0; i < len; i++) {
        if (str[i] >= '0' && str[i] <= '9') {     // 計算每個部份的數字
            temp_sum = temp_sum * 10 + (str[i] - '0');
        }
        else {
            sum += temp_sum;
            temp_sum = 0;
        }
    }

    printf("%s\n", str);
    printf("Ans=%d\n", sum);
    return 0;
}
